Portofolio

1. Portofolio Pribadi Interaktif

Buat sebuah situs web portofolio pribadi yang menampilkan proyek, keterampilan, dan pengalaman Anda. Fitur: Halaman beranda dengan animasi menarik. Bagian tentang diri Anda (About Me). Daftar proyek dengan gambar, deskripsi, dan tautan ke repositori atau demo. Formulir kontak dengan validasi. Responsif untuk berbagai perangkat. Teknologi yang digunakan: HTML5, CSS3, dan JavaScript. Framework CSS seperti Bootstrap atau Tailwind CSS. Library animasi seperti GSAP atau AOS. React atau Vue.js untuk SPA (Single Page Application).

2. Aplikasi To-Do List

Buat sebuah aplikasi to-do list yang memungkinkan pengguna menambahkan, menghapus, dan menandai tugas sebagai selesai. Fitur: Tambah, edit, dan hapus tugas. Filter tugas berdasarkan status (selesai/belum selesai). Simpan data tugas di local storage. Desain responsif untuk berbagai perangkat. Teknologi yang digunakan: HTML5, CSS3, dan JavaScript. React atau Vue.js untuk pengelolaan state dan komponen. Library UI seperti Material-UI atau Ant Design.

3. Dashboard Analitik

Buat sebuah dashboard interaktif yang menampilkan data analitik dalam bentuk grafik dan tabel. Fitur: Grafik interaktif (bar chart, line chart, pie chart) menggunakan data dummy. Filter data berdasarkan berbagai parameter (tanggal, kategori, dll.). Tabel data yang dapat diurutkan dan difilter. Desain responsif untuk berbagai perangkat. Teknologi yang digunakan: HTML5, CSS3, dan JavaScript. Framework CSS seperti Bootstrap atau Tailwind CSS. Library chart seperti Chart.js atau D3.js. React atau Angular untuk pengelolaan state dan komponen.

4. E-Commerce Store

Buat sebuah situs web e-commerce sederhana dengan daftar produk, keranjang belanja, dan halaman checkout. Fitur: Halaman daftar produk dengan pencarian dan filter. Halaman detail produk dengan gambar dan deskripsi. Keranjang belanja dengan fitur tambah/hapus produk. Halaman checkout dengan formulir pengisian data pengguna. Desain responsif untuk berbagai perangkat. Teknologi yang digunakan: HTML5, CSS3, dan JavaScript. Framework CSS seperti Bootstrap atau Tailwind CSS. React atau Vue.js untuk pengelolaan state dan komponen. Library state management seperti Redux atau Vuex.
